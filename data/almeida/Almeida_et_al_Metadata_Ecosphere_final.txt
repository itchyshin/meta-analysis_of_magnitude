The following R script files contain all analyses run in Almeida et al. (available in a Dryad Data Repository)

[a] Almeida_et_al_Meta-Analysis_direction-magnitude_code_Ecosphere_final.R
[b] Almeida_et_al_Meta-Analysis_environmental-conditions_code_Ecosphere_final.R


Information about the code:
+ The first script file [a] contains the code used to run meta-regressions for the full dataset as well as for the Condition-only, Reproduction-only, and Insect-only analyses.
  This includes evaluating sources of heterogeneity, comparing models with likelihood ratio tests, model diagnositcs, and examining publication bias. 
+ The second script file [b] contains the code used to run meta-regressions related to the effect of environmental conditions (magnitude, duration, recovery) on the absolute value of the effect size for the full dataset as well as for the Condition-only, Reproduction-only, and Insect-only analyses.
+ Both scripts contain code for plotting related figures.
+ Both scripts contain additional information about analyses in comments throughout.

Information about the data:
+ All of the above data files are called into their respective R script files.
+ Data that was extracted from the literature for the meta-analysis is contained in Almeida_et_al_Data_Ecosphere_final.csv. Details about variable names are below.
+ Correlations between species used to account for phylogenetic relationships are contained in the .csv files with "_Phylo_correlations_" in their names.
+ Each of the correlation data files represents a different subset of species that were included in the respective analyses. 
+ Covarance matrixes for each respective dataset are used to account for data extracted in experiments with shared controls.




Variables included in Almeida_et_al_Data_Ecosphere_final.csv
> ES.ID: A unique value for each effect size calculated. Used as a random effect in meta-regressions and to link the correlation for effect sizes representing multiple responses.
> Study: The identifier for each paper from which data was extracted and used in the meta-analysis. This typically included author names and year of publication. Used as a random effect in meta-regressions.
> Year: Year of publication of the study used in the meta-analysis. Not directly used in analyses.
> Category: The type of latent response extracted including those related to: condition, reproduction, longevity, learning, and mobility. Used as a fixed effect in meta-regressions.
> Sub_category: A narrower classification of latent response used to help parse responses within the broader categories, specifically within the condition and reproduction analyses. Used as a fixed effect in meta-regressions.
> Response: A more precise description of the type of latent response measured in the study. Not directly used in analyses.
> same_trt: A series of letters indicating if the responses extracted were from replicates exposed to the same treatment, with the same letter indicating that the same treatment was used. Used in preliminary analyses to ensure that any bias due to multiple effect sizes being extracted from the same treatment was accounted for.
> experiment: A series of letters indicating if the responses extracted were from replicates exposed to the same experiment, with the same letter indicating that the same experiment was used. Used in preliminary analyses to ensure that any bias due to multiple effect sizes being extracted from the same experiment was accounted for.
> Ctrl-shared_YN: A yes (Y) or no (N) indicator of if the calculated effect size used a control that was used to calculate another effect size (i.e., shared control). Used in preliminary analyses to ensure that effect sizes with shared controls were identified.
> LaterDiet: The diet that individuals across all treatments were provided after the nutritional deprivation period ended. Not direclty used in analyses.
> TrtFood: The diet that individuals were provided in the nutritional deprivation treatment. Not directly used in analyses.
> TrtResponse: The mean response of replicates within the nutritional deprivation treatment. Units varied depending on the type of response that was measured and they were sometimes recorded in the column "Response". Used to calculate Hedges' g and is renamed to "m2i" in the R code.
> TrtSE: The standard error of the mean from replicates within the nutritional deprivation treatment. Not directly used in analyses, but was sometimes used to calculate standard deviation.
> TrtSD: The standard deviation of the mean from replicates within the nutritional deprivation treatment. Used to calculate Hedges' g.
> CtrlFood: The diet that individuals were provided in the control treatment. Not directly used in analyses.
> CtrlResponse: The mean response of replicates within the control treatment. Units varied depending on the type of response that was measured and they were sometimes recorded in the column "Response". Used to calculate Hedges' g and is renamed to "m1i" in the R code.
> CtrlSE: The standard error of the mean from replicates within the control treatment. Not directly used in analyses, but was sometimes used to calculate standard deviation.
> CtrlSD: The standard deviation of the mean from replicates within the control treatment. Used to calculate Hedges' g and is renamed to "sd1i" in the R code.
> nTrt: The sample size (number of replicates) in the nutritional deprivation treatment. Used to calculate Hedges' g and is renamed to "n2i" in the R code.
> nCtrl: The sample size (number of replicates) in the control treatment. Used to calculate Hedges' g and is renamed to "n1i" in the R code.
> Species: The latin name of the species from which the responses were measured. Used as a random effect that is linked to the phylogenetic relationships between species (via the correlation matrices) as well as a random effect without considering the phylogenetic relationships.
> Sp_group: The broader taxonomic group that the species is grouped within including: amphibian, bird, decapod, echinoderm, fish, gastropod, insect, mammal, reptile. Used as a fixed effect in meta-regressions.
> age_treated: The early-life stage during which nutritional deprivation occured including: youngest or juvenile. Used as a fixed effect in meta-regressions.
> sex: The sex of the individuals included in the treatment, including: female, male, or mixed. Used as a fixed effect in meta-regressions.
> Trt_magnitude: The proportion of food provided to individuals in the nutritional deprivation treatment compared to individuals in the control treatment. The value provided in the datasheet is subtracted from 1 in the R script to ensure that a value of 1 indicates starvation and a value of 0 indicates that the same amount of food was provided for the nutrtional deprivation treatment and the control. Used as a fixed effect in meta-regressions.
> Trt_duration: The proportion of the time that treatment individuals were deprived food compared to the average length of time that control individuals spent in that life stage. The value of 1 indicates individuals were deprived for the entire lifestage, a value of 0 indicates they were not deprived at all. Used as a fixed effect in meta-regressions.
> Recovery: The proportion of time recovering after nutritional deprivation until the time (i.e., age in days) that the response was measured. The value of 1 indicates that the responses were measured right before the end of the lifespan for a non-deprived individual, a value of 0 indicates that responses were measured immediately after nutritional deprivation ended.
> LoH: A binary indicator of whether the response is more advantageous for an individual if it is higher (e.g., body size) or lower (e.g., development rate). Used to ensure that positive Hedges' g values would be associated with a benefit to fitness across all latent responses.
